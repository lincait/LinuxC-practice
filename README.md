# LinuxC编程代码

## 关于本仓库

本仓库是学习 Linux C 编程过程中的所有练习代码。
所有程序均在 Ubuntu 环境下编译通过，持续更新中。

## 目录结构

- `01_C_basics/` - C语言基础练习
    - `io_test.c` - 输入输出练习
    - `ctrl_test.c` - 流程控制练习
    - `array.c` - 数组练习
    - `pointer.c` - 指针练习
    - `Makefile` - 编译脚本

## 环境说明

- 操作系统：Ubuntu 22.04
- 编译器：GCC
- 编辑工具：VS Code + Remote-SSH
- 版本管理：Git + GitHub
